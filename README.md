# ABRA-Tools

## Frontend / Log Aggregator

### Frontend: React w/ TS built with Vite
How to build:
1. Clone repository
2. ```cd frontend/```
3. ```npm i```
4. ```npm run build```

### Backend / Log Aggregator: FastAPI
How to build:
1. ```cd backend/```
2. ```python3 -m venv venv```
3. ```. venv/bin/activate```
4. ```pip install -r requirements.txt```
5. ```python3 main.py```
